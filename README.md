# My Skills

A personal collection of Claude Code skills built by Masha Zolkipli.

## Skills

### `/daily-checklist-masha`
Masha's personalized morning briefing. Pulls live data from Google Calendar, HubSpot, and Jira to start the day with full context.

### `/response-draft`
Draft a customer support response for a ticket or request from HubSpot, Jira, or Slack.

## How to Use

1. Open this folder in Claude Code
2. Run any skill by typing `/skill-name` in the chat

## Setup

Make sure the following MCP connectors are configured in Claude Desktop:
- Google Calendar
- HubSpot
- Jira
- Slack
